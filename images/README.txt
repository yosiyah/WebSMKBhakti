📝 CARA MENAMBAHKAN GAMBAR:
==========================

1. Pilih folder yang sesuai dengan jenis gambar
2. Copy gambar ke folder tersebut
3. Rename file dengan nama yang deskriptif
4. Update HTML dengan mengganti src gambar

Contoh:
<!-- Sebelum -->
<img src="placeholder.jpg" alt="Lab Komputer">

<!-- Sesudah -->
<img src="images/facilities/lab-komputer.jpg" alt="Laboratorium Komputer Modern">

⚡ TIPS OPTIMASI:
================

✅ Format gambar: JPG, PNG, WebP
✅ Ukuran file: Max 500KB per gambar
✅ Resolusi: Sesuai rekomendasi di atas
✅ Nama file: gunakan lowercase dan dash (-)
✅ Alt text: Deskriptif dan informatif

🛠️ TOOLS YANG DIREKOMENDASIKAN:
==============================

- TinyPNG: Kompresi gambar (tinypng.com)
- ImageOptim: Optimasi gambar (imageoptim.com)
- Photoshop/GIMP: Edit gambar
- Canva: Buat gambar sederhana
